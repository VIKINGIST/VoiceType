# VoiceType

Локальний голосовий ввід для Windows. Натисни хоткей — говори — текст з'являється в будь-якому додатку. Повністю офлайн, безкоштовно, на твоїй GPU.

🇬🇧 [English version](README.md)

## Можливості

- **Голосовий ввід** — натиснув хоткей, сказав, текст вставився в активне поле
- **Три режими запису** — автоматичний (тиша = стоп), утримання клавіші, ручний перемикач
- **Подвійний хоткей** — основний хоткей для чистої транскрипції + хоткей обробки для фільтрованого/перекладеного результату
- **Пайплайн постобробки** — 5 вбудованих фільтрів (пунктуація, слова-паразити, повтори, абзаци, офіційний тон) + довільний промпт + LLM-переклад
- **LLM-провайдери** — DeepSeek або OpenRouter для постобробки (потрібен API key, опціонально)
- **33 мови розпізнавання** — українська, англійська, німецька, польська, французька, іспанська та ще 27 + авто-визначення
- **33 мови перекладу** — переклад результату через LLM на будь-яку підтримувану мову
- **Повністю офлайн** — модель Whisper на локальній GPU, дані нікуди не відправляються (постобробка опціонально використовує зовнішній LLM API)
- **GPU прискорення** — ~0.2с на розпізнавання 10 секунд аудіо (NVIDIA CUDA)
- **Неблокуючий пайплайн** — транскрипція виконується у фоновому потоці, інтерфейс не зависає
- **Вивантаження моделі при простої** — VRAM звільняється після налаштованого таймауту, модель перезавантажується при наступному використанні
- **Системний трей** — іконка з кольоровою індикацією стану + швидке підменю вибору мови перекладу
- **Підтримка кириличних хоткеїв** — рекордер хоткеїв працює з будь-якою розкладкою клавіатури
- **Сучасні налаштування** — dark theme GUI з 5 вкладками: Запис, Модель, Аудіо, Поведінка, Обробка
- **i18n** — інтерфейс 6 мовами
- **Інсталятор** — повноцінний Windows Setup з ярликами та деінсталятором

## Вимоги

- **Windows 10/11** (64-біт)
- **NVIDIA GPU** з підтримкою CUDA (GTX 1060+), мінімум **2 GB VRAM**
- Драйвер NVIDIA встановлений (перевірка: `nvidia-smi` в терміналі)
- Працює і на CPU, але в 10-50 разів повільніше
- Для постобробки: API key DeepSeek або OpenRouter (опціонально)

## Встановлення

### Інсталятор (рекомендовано)

1. Завантажити **[VoiceType-Setup.exe](https://github.com/VIKINGIST/VoiceType/releases)** (~1.3 GB)
2. Запустити інсталятор, обрати папку
3. Обрати опції: ярлик на робочому столі, Start Menu, автозапуск
4. Готово

> При першому запуску програма запропонує завантажити модель Whisper (~1.5 GB) з інтернету. Наступні запуски — з кешу, старт за 5-10 секунд.

## Як користуватися

### Режим "Автоматичний" (за замовчуванням)

1. Натисни `Alt+Q` — почнеться запис (іконка червона, біп)
2. Говори
3. Замовкни на 2 секунди — запис зупиниться автоматично
4. Текст вставиться в активне поле

### Режим "Утримання"

1. Затисни `Alt+Q` — запис поки тримаєш
2. Відпусти — стоп і розпізнавання

### Режим "Перемикач"

1. Натисни `Alt+Q` — старт
2. Натисни `Alt+Q` ще раз — стоп

### Хоткей обробки (опціонально)

Якщо налаштований хоткей обробки (Settings → Запис), він працює так само як основний хоткей, але перед вставкою результат проходить через пайплайн постобробки:

- Застосовуються активні фільтри (пунктуація, прибирання слів-паразитів тощо)
- Якщо заданий довільний промпт — він надсилається до LLM
- Якщо налаштований переклад — текст перекладається на обрану мову

Хоткей обробки потребує налаштованого LLM-провайдера та API key. Без них він поводиться як звичайний хоткей.

Налаштування: **правий клік на іконку в треї → Settings**

### Меню трею

- **Settings** — відкрити налаштування (якщо вже відкриті — вікно виходить на передній план)
- **Translate to** — підменю швидкого вибору мови LLM-перекладу
- **Quit** — зупинити запис і вийти

## Налаштування

### Запис
| Параметр | За замовчуванням | Опис |
|---|---|---|
| Мова інтерфейсу | English | 6 мов на вибір. Потребує перезапуску. |
| Гарячa клавіша | `alt+q` | Кнопка "Record" для захоплення комбінації. Підтримує будь-яку розкладку, включно з кириличною. |
| Хоткей обробки | (немає) | Те саме що основний хоткей, але результат проходить постобробку |
| Режим запису | Автоматичний | Автоматичний / Утримання / Перемикач |
| Мова розпізнавання | Авто-визначення | 33 мови + авто |

### Модель
| Параметр | За замовчуванням | Опис |
|---|---|---|
| Модель Whisper | large-v3-turbo | Найкращий баланс швидкості та якості |
| Пристрій | GPU (CUDA) | GPU або CPU |
| Точність | float16 | float16 для GPU, float32 для CPU |
| Таймаут вивантаження | 5 хв | Вивантажити модель з VRAM після простою. 0 = не вивантажувати. |

| Модель | VRAM | Швидкість (10с аудіо) | Якість |
|---|---|---|---|
| **large-v3-turbo** ★ | ~1.6 GB | ~0.2с | Висока |
| large-v3 | ~3.0 GB | ~1.0с | Найвища |
| medium | ~1.0 GB | ~0.1с | Середня |
| small | ~0.5 GB | ~0.05с | Базова |

Моделі завантажуються з HuggingFace при першому використанні. Управляти кешем (завантаження, видалення) можна у вкладці Модель.

### Аудіо
| Параметр | За замовчуванням | Опис |
|---|---|---|
| Мікрофон | Системний за замовчуванням | Вибір пристрою введення |
| Чутливість VAD | 0.50 | Менше = більш чутливий до голосу |
| Тиша для зупинки | 2.0с | Тривалість тиші для авто-стопу (Автоматичний режим) |
| Макс. тривалість | ∞ | Ліміт у секундах (0 = без ліміту) |

### Поведінка
| Параметр | За замовчуванням | Опис |
|---|---|---|
| Авто-вставка | Вкл | Ctrl+V після розпізнавання |
| Копіювати в буфер | Вкл | Копіює текст в clipboard |
| Нотифікації | Вкл | Toast з розпізнаним текстом |
| Звук старту/стопу | Вкл | Звукові сигнали |
| Автозапуск з Windows | Викл | Запуск при старті системи |

### Обробка
| Параметр | За замовчуванням | Опис |
|---|---|---|
| LLM-провайдер | DeepSeek | DeepSeek або OpenRouter |
| API key | (порожньо) | Твій API key. "Test" — перевірити, "Register" — відкрити сайт провайдера. |
| Перекласти на | (немає) | Перекласти результат на цю мову через LLM (33 мови) |
| Пунктуація | Викл | Додати пунктуацію та великі літери |
| Прибрати слова-паразити | Викл | Прибрати "ну", "як би", "е-е" тощо |
| Прибрати повтори | Викл | Прибрати повторювані слова та фрази |
| Абзаци | Викл | Розбити на абзаци за темою |
| Офіційний тон | Викл | Переписати у формальному стилі |
| Довільний промпт | (порожньо) | Вільна інструкція для LLM, виконується після фільтрів |

Фільтри та переклад застосовуються тільки при використанні **хоткея обробки**. Основний хоткей завжди дає чисту транскрипцію.

Всі налаштування також доступні у `config.yaml` (поряд з виконуваним файлом).

## Вирішення проблем

| Проблема | Рішення |
|---|---|
| "Whisper Error" при старті | Потрібен інтернет для завантаження моделі (~1.5 GB). Перевір VRAM: `nvidia-smi`. Спробуй модель `medium` або `small`. |
| Хоткей не працює | Переконайся що VoiceType запущено (іконка в треї). Запусти від адміністратора для додатків з підвищеними правами. |
| Текст не вставляється | Курсор має бути в текстовому полі. Текст завжди копіюється в буфер — спробуй Ctrl+V вручну. |
| Погано розпізнає | Говори чіткіше, ближче до мікрофона. Перевір мікрофон у налаштуваннях. Спробуй `large-v3`. |
| "Вже запущено" | Закрий через трей → Вихід. |
| Хоткей обробки не реагує | Перевір що LLM-провайдер та API key задані в Settings → Обробка. Використай "Test" для перевірки ключа. |
| LLM-переклад повільний | Виклики DeepSeek/OpenRouter API додають 1-5 секунд залежно від довжини тексту та швидкості мережі. |
